= 支持的NetApp Data Classification实例类型
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
NetApp Data Classification 软件必须在满足特定操作系统要求、RAM 要求和软件要求的主机上运行。在云中部署 Data Classification 时，我们建议您使用具有"大"特征的系统来实现全部功能。

在下表中，如果标记为"默认"的系统在安装 Data Classification 的区域中不可用，则将部署表中列出的下一个实例类型。



== AWS 实例类型

[cols="20,30,25"]
|===
| 系统大小 | 规格 | 实例类型 


| 特大号 | 32 个 CPU、128 GB RAM、1 TiB gp3 SSD | https://aws.amazon.com/ec2/instance-types/m6i/["m6i.8xlarge"^]（默认） 


| 大型 | 16 个 CPU、64 GB RAM、500 GiB SSD | https://aws.amazon.com/ec2/instance-types/m6i/["m6i.4xlarge"^]（默认）m6a.4xlarge m5a.4xlarge m5.4xlarge m4.4xlarge 
|===


== Azure 实例类型

[cols="20,30,25"]
|===
| 系统大小 | 规格 | 实例类型 


| 特大号 | 16 个 CPU、64 GB RAM、操作系统磁盘（2,048 GiB，最低 250 MB/s 吞吐量）和数据磁盘（1 TiB SSD，最低 750 MB/s 吞吐量） | https://learn.microsoft.com/en-us/azure/virtual-machines/sizes/general-purpose/dv5-series["Standard_D32s_v5 或 Standard_D32_v5"^] 


| 大型 | 16 个 CPU、64 GB RAM、500 GiB SSD | link:https://learn.microsoft.com/en-us/azure/virtual-machines/sizes/general-purpose/dsv5-series?tabs=sizebasic["Standard_D16_v5 或 Standard_D16s_v5"^] (默认) 
|===


== GCP 实例类型

[cols="20,30,25"]
|===
| 系统大小 | 规格 | 实例类型 


| 大型 | 16 个 CPU、64 GB RAM、500 GiB SSD | https://cloud.google.com/compute/docs/general-purpose-machines#n2_machines["n2-标准-16"^]（默认）n2d-standard-16 n1-standard-16 
|===
